JAVA-GLASSFISH

LENGUAJE QUE USA: JAVA

GlassFish es un servidor de aplicaciones de código abierto desarrollado por Oracle que implementa las especificaciones de Java Enterprise Edition (Java EE). Sus características principales incluyen:
1. Implementación de Java EE.
2. Administración y despliegue sencillos.
3. Escalabilidad para diferentes cargas de trabajo.
4. Interoperabilidad con otros sistemas.
5. Soporte para microservicios.
6. Herramientas para el desarrollo ágil.
7. Funcionalidades de seguridad integradas.
8. Compatibilidad con implementaciones en la nube.

Empresas que utilizan GlassFish:
1. Oracle
2. Sun Microsystems
3. Red Hat
4. Apache Software Foundation
